Java Interpreter for Brainf*** Programs

Un breve experimento con un lenguaje minimalista.

This is a simple interpreter built in Java for an esoteric programming language with an... unfortunate name.

This language was created in 1993 by Urban Müller. It operates on a one-dimensional array of cells, similar to a Turing machine, and consists of only eight instructions:

  • > - Move the pointer to the next cell.
  • < - Move the pointer to the previous cell.
  • + - Increment the value at the pointer by 1.
  • - - Decrement the value at the pointer by 1.
  • . - Print the value at the pointer as an ASCII character.
  • , - Wait for the user's input and write it to the cell at the pointer.
  • [ - Skip to the matching ] instruction if the value at the pointer is zero.
  • ] - Rewind to the matching [ instruction if the value at the pointer is not zero.

This implementation has 32,768 8-bit cells, and the source code can be downloaded from here.
